Administrador: Control de viajes, módulos de ventas, lógistica, compras, mantenimiento, inventarios y más de 30 reportes en vista, pdf y excel. Estaba como administrador de la plataforma, levantaba requerimientos para actualizaciones, análisis, diseño, programacion y despliegue en servidor.

Cancelación: Requisiciones de Compra, Ordenes de Compra, Ordenes de Compra.

* Aplicación Web - Reportes Checador

Creación de plataforma para recursos humanos, donde consultan las checadas de los empleados, los agrupa por empleado, descarga un documento de excel para ser enviado a contabilidad; Sea semanal, quincenal o mensual. La aplicación se conecta a un sistema de almacenamiento de checadas BioTime de la empresa ZKTeco, se configuro totalmente en un ambiente de producción local.

* Aplicación Web - Modulo de Arenas

Creación de plataforma para el manejo de ventas de Arena para Gato que se le vende a los clientes: Walmart México, Heb México y Dollar General USA. Para los clientes de Walmart y Heb, se crearon modulos para cargar las ordenes de compra en formato Excel (csv), se leen los documentos, para despues crear las ordenes de venta usando SAP-API, y por ultimo creando el pedido de viaje en la plataforma de Transportes, por ultimo se reciben los avisos de pago, se cargan al sistema y se calcula el pago de acuerdo a ciertos descuentos que se aplican (factor logistico, penalizaciones y descuentos generales). Para Dollar General, se reciben las ordenes de compra, se crean los documentos de ordenes de venta en SAP y para la logistica se contrata una empresa internacional (Expeditor), la plataforma lleva el control de los pedidos, la monitorización de los viajes y la recepción de los documentos de pago.

* Aplicación Web - Reportes Proeco

Se crearon reportes ejecutivos para monitorear la operación de Proeco: Entradas de materia prima, producciones de artículos estrella, salidas de producto terminado, control de inventarios, reporte integrador master.
Para esta plataforma hice la alta de requerimientos con el gerente de Proeco, gerente de facturación, gerente financiero, gerente de producción, contadores y jefe de procesos ecológicos.
La plataforma se alimenta de una base de datos en SQL Server de SAP Business One, para las producciones, se instalo un PLC con proveedor externo, contando con una API publica para el monitoreo de los tanques de materia prima y producto terminado, estas lecturas crean registros en una base de datos local, arrojando los registros de cada corrida, estos se revisa por el jefe de producción, si es aprobada se usa una API para crear la producción en SAP; Todo este sistema se penso para que las personas encargadas de la operación no tiendan a equivocarse al capturar la información evitando el error humano.

* Unifi - Creación de Red Local

Se creo una red local para el area de Sistemas, area de administrativos, area de mantenimiento, area de vigilancia y una conexión directa para el equipo de la bascula. Se usaron los Access Point de la marca Unifi. Para el area de sistemas y administrativos se creo una red de malla. para mayor cobertura por las caracteristicas del area.
